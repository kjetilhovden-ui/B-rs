# Instruksjoner for Claude Code i dette prosjektet

- **Alltid svar på norsk.** Brukeren er nybegynner på koding og foretrekker
  norsk i all kommunikasjon (forklaringer, statusoppdateringer, commit-
  meldinger der det er naturlig).
- **Opprett alltid en pull request automatisk** når en avgrenset del av
  implementasjonen er fullført og pushet, uten å spørre om lov først. Dette
  gjelder spesifikt dette repoet — brukeren har bekreftet dette som en
  stående regel (ikke bare for én enkelt gang).
- Planen for prosjektet (faser, arkitekturvalg, begrunnelser) ligger i
  `/root/.claude/plans/prosjekt-oslo-b-rs-case-finner-valiant-pie.md`.
